Результаты тестирования

Проводим тестирование интернет-ресурса: https://job.ozon.ru/courier/?utm_source=organic&utm_campaign=marketplace

Среда окружения:

  • OS: Майкрософт Windows. Версия 22H2 (сборка ОС 22621.1555)
  • Браузер: Google Chrome версия 112.0.5615.138
  • Программа экранного доступа: nvda. Версия: 2022.3.3

В процессе аудита интернет-ресурса на доступность были обнаружены следующие ошибки:

Уровень : блокирующие.

Не явная роль у элементов с выпадающим списком выбора

Нарушение: Положение 4.1.3 Адаптируемость (Критерий 3 Характеристики, воспринимаемые органами чувств)

расположение: Элементы формы "Оставьте заявку и мы с вами свяжемся"
  • Гражданство
  • Пол
  • Город
Результаты

Ожидается, что эти поля формы для программ экранного доступа будут определяться как combobox или select с выбором одного элемента из выпадающего списка

Но, данные элементы формы, при фокусировании определяются как однострочные редакторы, которые подразумевают ввод текста и в которые нельзя вводить текст.
Список раскрывается с элементами для выбора при активации на подпись к данному полю. Данная текстовая подпись не принимает фокус, что для незрячего пользователя не приемлимо и пользователь не будет понимать как с данным элементом взаимодействовать.
